Detailed Description

These macros are available to help make debugging easier.

Macro Definition Documentation

◆ ZAKERO_YARS__DEBUG_DISABLED

#define ZAKERO_YARS__DEBUG_DISABLED   false

The inverse of ZAKERO_YARS_DEBUG_ENABLED and is used to enable or disable the debugging messages.

◆ ZAKERO_YARS_DEBUG

#define ZAKERO_YARS_DEBUG
Value:
if(ZAKERO_YARS__DEBUG_DISABLED) {} \
else std::cerr \
<< __FILE__"(" \
<< std::to_string(__LINE__) \
<< ") "__PRETTY_FUNCTION__

If the ZAKERO_YARS_ENABLE_DEBUG macro is enabled, this macro will write to the std::cerr stream. Nothing will happen when the macro is disabled.

Example

ZAKERO_YARS_DEBUG << "Insert debugging message here.\n";

◆ ZAKERO_YARS_DEBUG_VAR

#define ZAKERO_YARS_DEBUG_VAR (   var_)    ZAKERO_YARS_DEBUG << #var_ << ": " << var_ << "\n";

Does the samething as ZAKERO_YARS_DEBUG but is specialized to work with std::ostream compatible objects.

Example

uint64_t foo = 42;
ZAKERO_YARS_DEBUG_VAR(foo);
