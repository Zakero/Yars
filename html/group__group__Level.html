Detailed Description

Level plays an important role in Y.A.R.S. because it determines how strong the character's attributes and skills are. The following aspects make up a Level:

  • Value
    This is the raw value of a Level, such as "Skill is Level 5". When a Yars::Skill or an Yars::Attribute refers to a Level, this is the value that is used.

  • Partial
    When a Level increases, the amount of the increase is a floating point number. The partial values accumulate the level increases until the amount reaches the threshold to increase the value.

  • Modifier
    The modifier is a bonus or penalty that is applied to the value.

The requirements for a level increase is: \(Level.partial>=Level.value\_base\) When the requirements are satisfied, Level.partial is decreased by Level.value_base then the Level.value_base is increased by 1. Finally, the Level.modifier is applied and the result is stored in Level.value.

The reason that the Level.modifier is applied to Level.value_base and not Level.partial is to maintain the effectiveness of the modifier. The higher the Level.value is, the less likely it is that an increase to Level.partial will matter. Applying the modifier to the value, a +1 is useful to all levels.

The Level.modifier only takes affect when the Level.value_base is greater than 0. The Level.value will never go below 0, regardless of the Level.modifier value.

Increase the provided level with the specified amount. If amount is less than 0, then the method will just return 0.

Example

Yars::Level level;
Yars::levelIncrease(level, 1.5f);
